# NovaCRM — Product FAQ

## What is NovaCRM?
NovaCRM is a cloud-based customer relationship management platform designed for small and mid-size businesses. It helps teams manage contacts, track deals, automate workflows, and analyze sales performance.

## What features does NovaCRM include?
NovaCRM includes contact management, deal pipeline tracking, email integration, task automation, reporting dashboards, team collaboration tools, and a REST API for custom integrations.

## Does NovaCRM support email integration?
Yes. NovaCRM integrates with Gmail, Outlook, and any IMAP/SMTP email provider. Emails are automatically linked to the relevant contact or deal record.

## Can I import data from another CRM?
Yes. NovaCRM supports CSV import for contacts, deals, and notes. We also offer guided migration from Salesforce, HubSpot, and Pipedrive.

## Is there a mobile app?
Yes. NovaCRM has native iOS and Android apps with full access to contacts, deals, tasks, and notifications. Offline mode is available for viewing cached records.

## Does NovaCRM have an API?
Yes. NovaCRM provides a RESTful API with OAuth 2.0 authentication. API documentation is available at docs.novacrm.example.com. Rate limits are 1,000 requests per minute on paid plans.

## What browsers are supported?
NovaCRM supports the latest versions of Chrome, Firefox, Safari, and Edge. Internet Explorer is not supported.

## Can I customize the deal pipeline stages?
Yes. You can create unlimited custom pipeline stages, rename them, reorder them, and set automation rules for each stage transition.

## Does NovaCRM support multiple currencies?
Yes. NovaCRM supports 50+ currencies with automatic exchange rate updates. Each deal can be assigned a different currency.

## Is there a sandbox or test environment?
Yes. All Business and Enterprise plans include a sandbox environment for testing workflows and API integrations without affecting production data.

## How do I contact support?
You can reach support via the in-app chat widget, email at support@novacrm.example.com, or by calling 1-800-NOVA-CRM during business hours (9 AM – 6 PM EST, Monday–Friday).
